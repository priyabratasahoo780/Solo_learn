## 🌟 Welcome to SoloLearn
SoloLearn is not just another e-learning platform; it's a meticulously crafted digital ecosystem designed for the modern developer. Built with a **Dark-Mode First** philosophy, it combines sleek aesthetics with powerful functionality, including an **AI-driven Code Tutor** and automated certification.

---

## 🤖 Meet Your AI Code Tutor
> "Accelerate your learning with a personalized AI mentor that never sleeps."

The **AI Code Tutor** is integrated directly into your learning flow. It's not just a chatbot; it's a context-aware programming expert.

### ✨ What can the AI Tutor do?
- **Code Explanation**: Paste any snippet, and get a line-by-line breakdown.
- **Debugging Assistance**: Stuck on a bug? The AI helps you identify and fix it in real-time.
- **Interactive Quizzes**: Generate personalized tests based on the topic you're studying.
- **Multi-lingual Support**: Expert knowledge in **JavaScript, React, Python, SQL, Git, and CSS**.

## 🔥 Key Features

#### 🌙 Premium Dark Mode UI
Experience a stunning interface built with **Tailwind CSS** and **Framer Motion**. Smooth transitions, neon accents, and a layout that feels alive.

#### 🎓 Professional Certification
Complete a course and instantly receive a high-quality PDF certificate. Generated by **PDFKit** on the backend and delivered straight to your email.

#### 🛡️ Advanced Security & Anti-Inspection
We take content protection seriously. Our custom security layer includes:
- **Global DevTools Blocking**: Prevents inspection via keyboard shortcuts and browser menus.
- **Right-Click Protection**: Stops unauthorized content copying.
- **Intelligent Redirection**: Automatically redirects suspicious activity to safety.

#### 📊 Real-time Progress Tracking
Visualize your learning journey with interactive charts and a global leaderboard. Built with **Recharts** for pixel-perfect data representation.

## 🛠️ The Tech Stack

### Frontend Architecture
- **React 19 & Vite**: The fastest foundation for modern web apps.
- **Framer Motion**: Powering fluid, cinematic animations.
- **Lucide React**: Beautiful, consistent iconography.
- **Tailwind CSS**: A utility-first CSS framework for custom, premium styling.

### Backend Infrastructure
- **Node.js & Express**: High-performance, scalable API server.
- **MongoDB (Mongoose)**: Document-based database for flexible data management.
- **JWT & Bcrypt**: Industry-standard authentication and password security.
- **Nodemailer**: Automated notification and certificate delivery system.

## 🚀 Getting Started

### 1. Clone & Install
```bash
git clone https://github.com/priyabratasahoo780/Solo_learn.git
cd Solo_learn
```

### 2. Configure Environment
Create a `.env` in the `backend/` directory:
```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_super_secret_key
EMAIL_HOST=smtp.gmail.com
EMAIL_PORT=587
EMAIL_USER=your_email@gmail.com
EMAIL_PASS=your_app_password
EMAIL_FROM=your_email@gmail.com
```

### 3. Launch the Platform
```bash
# In backend terminal
npm install
npm run dev

# In client terminal
npm install
npm run dev
```
