# 🎓 SoloLearn: The Elite Engineering Academy

![SoloLearn Premium Hero](./assets/hero_premium.png)

> **"The world's premier digital institution for technical mastery. Simulations. Battlegrounds. Certifications."**

SoloLearn is a high-fidelity, Neubrutalist-inspired educational platform designed for the modern engineer. Built on the **Engineer OS** design system, it synchronizes academic rigor with tactical gaming mechanics to create an immersive learning environment.

---

## 🚀 Key Departments

### 🧠 Academic Intel (AI Tutor)
Powered by the state-of-the-art **Gemini 2.5 Flash** model, our AI Tutor provides real-time, context-aware technical guidance.
- **Conversational Memory**: Remembers your previous queries for a seamless learning flow.
- **Tactical Quick Actions**: specialized protocols for *Deep Breakdowns*, *Code Reviews*, and *Architecture Analysis*.
- **Institutional Persona**: Operates under strict "Socratic" or "Rigorous" protocols to maximize retention.

### ⚔️ Strategic Duels (Battleground)
A real-time competitive arena where students authorize their expertise through high-stakes code challenges.
- **Persistent Lobby**: 10-second polling for stable global matchmaking.
- **Tactical HUD**: Minimalist, high-contrast interface designed for split-second decisions.

### 📼 Neural Sandbox (Mock Interviews)
AI-simulated interview protocols that prepare you for the highest levels of corporate recruitment.

## ⚙️ Installation & Deployment

### 1. Initialize Backend
```bash
cd backend
npm install
npm run dev
```

### 2. Initialize Client
```bash
cd client
npm install
npm run dev
```

### 3. Environment Configuration (`.env`)
Ensure the following tactical credentials are set in `/backend/.env`:
- `MONGO_URI`: Your MongoDB cluster connection string.
- `GEMINI_API_KEY`: Your authorized Gemini 2.5 API key.
- `JWT_SECRET`: Institutional security token.
